# Evaluate a news article with Natural Language Processing

* How to install all of dependencies file 
    > npm install

* Run in development mode 
    > npm run build-dev

* Start back-end server 
    >  npm run start

* Run in production mode 
    > npm run build-prod

* Start back-end server 
    > npm run start

* Languages used in this project :
    * HTML5
    * CSS3
    * JavaScript
    * SASS
    * Webpack
    * Node JS (Express)

* Testing :
    * Jest

* API :
    * Meaning Cloud API
